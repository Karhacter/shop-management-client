# Subject project in College
# How to run code ?
- Pull code in to your device
- run this command in terminal where you pull code
```
$ npm start
```
- when you finish then run one of these commnad
```
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
